Games

Frameland - A Binary Tale PC & Mac WIP

A Roleplaying Game told from two perspectives.

(click on the orange items for details)
  • Programmed a custom 2D-Engine tailored to the needs of a tilebased 2D-RPG.
  • Engine Features: Scenes, Tilemaps, Actions, Input Abstraction, Shaders, GUI, ...
  • Created a Turn-Based Battlesystem that requires timed-input for Attacks and Defends.
  • Additionally made Tools to quickly iterate on the game: Level Editor, Particle Editor & more
  • UI Design + Implementation for all Menus (cursor controlled)
  • An API that can be accessed by the Scripting System (or internally).
  • Smooth Character Movement (walks on the grid but can switch direction anytime)
  • Implemented RPG specific features: Levelling-System, Inventory, Weapons, Items, ...
  • Fast Collision Detection: Constant time regardless the amount of objects
  • Created On-Map enemies that implement swarm-like behaviour like seeking and avoiding
  • Data-Oriented Particle System that can draw up to 100000 particles on my computer (Macbook Pro 2015).
  • Made several smaller minigames
