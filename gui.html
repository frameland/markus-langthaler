The GUI System uses a hierarchical parent-child relationship for easier layout.
At the root there is GUI object that all other Widgets must be added to.
A Widget base class is extended by specific Widgets such as Buttons, Textboxes, etc.
You can also set a Listener to receive UI Events, such as 'Clicked', 'Checked', 'Value Changed'.


Full List of GUI-Widgets:

  • Button
  • Checkbox
  • Colorpicker (RGB and HSV modes)
  • Combobox
  • IconToggleList
  • ItemList
  • Label
  • Panel
  • Separator
  • Slider
  • SliderValue
  • Tabs
  • Textarea (supports code highlighting)
  • Textfield
